回调函数接收 DB的返回值

异步连接池维护的是长连接。

怎样实现异步的连接

同步连接池需要创建多个线程。

异步连接不需要创建额外的线程，复用主线程、复用网络模块。

  1. 创建一个非阻塞的连接
  2. 接入数据库的协议。若数据库提供异步的接口，则只需要适配事件的框架就可以了。
